# Portfolio-projekti 

## Projektin tarkoitus 

Tämän projektin tarkoituksena on toimia henkilökohtaisena portfoliona, jossa esittelen osaamistani ja kurssin aikana tehtyjä projekteja. 

## Kenelle sivu on tarkoitettu 

Sivu on tarkoitettu opettajalle, mahdollisille työnantajille sekä muille, jotka ovat kiinnostuneita osaamisestani.
